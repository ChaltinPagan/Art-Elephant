Art Elephant Routes

Base URL: http://localhost:8000

Users
Request Data Endpoint
GET All Users /users
POST New User /users/new
POST Return Single User /users/:email
POST Log in User /users
PUT Update User Password /users/update-pw
PUT Update User's Name and Email /users/update
Artists
Request Data Endpoint
GET All Artists /artists
GET Return Single Artist (by artist ID) /artist/:id
POST New Artist /artists/new
POST Return Artist by User ID /artists/:user_id
PUT Update Artist (by artist ID) /artists/:id
